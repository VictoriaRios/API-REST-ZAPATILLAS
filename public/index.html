API DE ZAPATILLAS

¿Sobre qué trata esta API?

Esta API está diseñada para gestionar información sobre zapatillas. Proporciona las siguientes operaciones para interactuar con los datos:

  • GET: Recupera la lista completa de zapatillas disponibles.
  • GET by ID: Obtiene información detallada de una zapatilla específica mediante su identificador único.
  • POST: Permite agregar nuevas zapatillas al sistema.
  • PUT: Actualiza la información de una zapatilla existente, identificada por su ID.
  • DELETE: Elimina una zapatilla del sistema utilizando su identificador único.

Esta API es ideal para aplicaciones que requieran gestionar un catálogo de zapatillas, como tiendas en línea, inventarios o sistemas de recomendación de productos.

GET /api/zapatillas

Permite recuperar una lista completa de zapatillas disponibles en la API, proporcionando información detallada de cada una para facilitar su consulta o visualización en aplicaciones externas.

GET BY ID /api/zapatillas/:id

Obtiene información específica de una zapatilla en particular, identificada mediante su ID único. Ideal para acceder a detalles puntuales de un producto.


POST /api/zapatillas

Permite agregar una nueva zapatilla al sistema. Se pueden enviar datos como nombre, precio y marca, que serán almacenados en la base de datos de la API.

Este es el modelo que espera recibir la API:

                    {
                        nombre: String,
                        precio: Number,
                        marca: String,
                    }
PUT /api/zapatillas/:id

Actualiza la información de una zapatilla existente en el sistema. Esta operación requiere un ID válido para identificar el registro que se desea modificar.

Este es el modelo se mostrará al momento de editar:

                    {
                        nombre: String,
                        precio: Number,
                        marca: String,
                    }
DELETE /api/zapatillas/:id

Elimina del sistema una zapatilla específica utilizando su ID. Esta acción es irreversible y asegura que el registro se elimine por completo.
